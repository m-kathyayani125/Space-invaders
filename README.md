
# Space Invaders

A Game built on Javascript and Canvas.



## Controls

1.Control Movements : 'A' key for going left and 'D' key for going right

2.Space bar for fire.

Note: You have to escape from the bullets of the Opponent, orlse GAME OVER !!!
